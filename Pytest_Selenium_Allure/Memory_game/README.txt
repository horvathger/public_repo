A memory game, egy webes alkalmazás automatizált megoldása Python és Selenium használatával.
A játék automatizálásához a DOM elemzésére volt szükség, hogy a legegyszerűbb megoldást megtaláljam.
A játék végén asserttel vizsgáltam meg az eredményt kijelző ablakot, mielőtt bezártam.

A memória játék az alábbi linken érhető el:
https://high-flyer.hu/selenium/memory-game.html
